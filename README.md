🌙 Dark Mode Toggle

A modern Dark Mode Toggle built using HTML, CSS, and Vanilla JavaScript. This project allows users to switch between light and dark themes with a single click.

The toggle enhances user experience by providing better readability in different lighting conditions. It demonstrates how to use JavaScript event handling and CSS variables to dynamically update the UI.

This project is ideal for beginners who want to learn how theme switching works in real-world applications such as dashboards, websites, and mobile-friendly interfaces.

✨ Features

🌙 Toggle between light & dark themes

⚡ Smooth transition effects

🎨 Clean and modern UI


🛠️ Tech Stack

HTML5 – Structure

CSS3 – Styling & transitions

JavaScript (ES6) – Toggle logic
